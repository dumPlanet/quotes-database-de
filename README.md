# quotes-database-de
A database in xml with a lot of german quotes for any purpose and free.

The structure follow this build

"<quotes>
    <quote>
        <id>1</id>
        <quotetext></quotetext>
        <author></author>
        <job></job>
        <advertising></advertising>
        <date></date>
    </quote>
  
  </quotes>"
